# Roma-STT — инструкция для пользователя

Простая программа: нажимаете горячую клавишу — говорите — нажимаете ещё раз, и сказанный текст вставляется в то место, где стоит курсор (в любом приложении). Работает полностью на вашем компьютере, без интернета после установки.

---

## Что нужно

- Компьютер с **Windows 10 или 11**.
- Один раз поставить несколько бесплатных программ (ниже написано, как это сделать автоматически или вручную).

---

## Первый запуск (по шагам)

1. **Откройте папку с программой.** Запустите **`roma-stt.bat`** двойным щелчком. Откроется окно с меню.

2. **Выберите пункт 1** (Установка) и нажмите Enter. Скрипт:
   - Проверит и установит через WinGet: uv, Git, CMake, Visual Studio Build Tools.
   - Если у вас NVIDIA — предложит установить CUDA Toolkit; если AMD — Vulkan SDK.
   - Создаст Python-окружение, скачает модель распознавания и соберёт программу.
   - Это займёт несколько минут (при первом запуске — дольше из-за сборки).

   > Если WinGet не установлен, откройте **PowerShell от имени администратора** и выполните:
   > ```powershell
   > $progressPreference = 'silentlyContinue'
   > Install-PackageProvider -Name NuGet -Force | Out-Null
   > Install-Module -Name Microsoft.WinGet.Client -Force -Repository PSGallery | Out-Null
   > Repair-WinGetPackageManager -AllUsers
   > ```
   > Затем закройте окно и снова запустите `roma-stt.bat`.

3. **Посмотрите в шапке меню строку «Готовность»**. Если написано **OK** — можно запускать службу. Если **не готово** — прочитайте краткую подсказку справа и при необходимости снова выполните пункт **1**. Полный отчёт: в командной строке из папки проекта выполните `uv run python scripts/check_ready.py`.

4. **Выберите пункт 2** (Запустить службу). Программа свернётся в трей (правый нижний угол экрана). Нажмите Enter, чтобы вернуться в меню.

5. **Как пользоваться:**
   - Откройте любое приложение, куда хотите вставить текст (Блокнот, браузер, Word и т.д.).
   - Поставьте курсор в нужное место.
   - Нажмите **Ctrl+F2** (запись, по умолчанию) — прозвучит **один короткий высокий сигнал**, начнётся запись. Говорите.
   - Нажмите **Ctrl+F3** (стоп, по умолчанию) — прозвучат **два коротких низких сигнала**, запись остановится, пойдёт распознавание, и текст вставится в то место, где стоит курсор.

6. **Чтобы выключить программу** — снова откройте **roma-stt.bat**, выберите пункт **3** (Остановить службу).

## Если что-то пошло не так

- **«winget не найден»** — установите WinGet вручную (см. шаг 2 выше), затем снова запустите пункт **1**.
- **«uv не найден»** после установки — закройте окно и откройте заново: PATH обновляется при новом запуске. Затем снова выберите пункт **1**.
- **В шапке «Готовность: не готово»** — выполните пункт **1** или исправьте то, что указано в подсказке; меню обновится (Enter).
- **«Отказано в доступе» при установке** — папку .venv использует другой процесс. Закройте Roma-STT в трее (пункт 3), все окна терминала из этой папки, Cursor/VS Code. Затем снова запустите пункт **1**.
- **«Горячая клавиша занята» при запуске** — другое приложение уже использует выбранную комбинацию. Используйте пункт **5** (Подбор свободной клавиши) или пункты **6/7** (задать вручную).
- **Программа не вставляет текст** — убедитесь, что курсор стоит в поле ввода. Горячие клавиши по умолчанию — **Ctrl+F2** (запись) и **Ctrl+F3** (стоп).

## Пункты меню коротко

- **1** — установка: без `.venv` — полная установка; с `.venv` — **1** обновить/переустановить, **2** удалить `.venv` и `models`.
- **2** — запустить службу (уйдёт в трей).
- **3** — остановить службу.
- **4** — модели распознавания: посмотреть список, скачать другую, выбрать активную.
- **5** — подбор свободной горячей клавиши (автоматический тест).
- **6** — горячая клавиша записи (по умолчанию Ctrl+F2).
- **7** — горячая клавиша стопа (по умолчанию Ctrl+F3).
- **8** — язык распознавания (ru, en и др.).
- **9** — устройство ввода (микрофон): сканировать и выбрать номер.
- **10** — выход из меню.

Уведомления Windows из трея: по умолчанию выключены; включить только в **config.yaml** (`notifications: true`).

Постобработка текста после распознавания **всегда включена** (отдельного пункта нет). Проверка готовности: `roma-stt.bat check`.

---

## Горячая клавиша

По умолчанию: **Ctrl+F2** — запись, **Ctrl+F3** — стоп. Поддерживаются F1–F24 и модификаторы Ctrl, Alt, Shift, Win.
Изменить клавиши можно в меню (пункты **6** и **7**) или прямо в файле **config.yaml**.
